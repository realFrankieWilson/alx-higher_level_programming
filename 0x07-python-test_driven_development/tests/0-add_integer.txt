###################################
DOCUMENTATION ON THE ADD FUNCTIONS#
==================================#

What the function does:
______________________
The function adds 2 integers
Returns a Sum of the two integer.




Importing the module:
_____________________
>>> add_integer = __import__('0-add_integer').add_integer
>>> 




USAGE:
======

Successful Usage
________________

We can use normal integers:
---------------------------
>>> add_integer(2, 1)
3
>>> 

We can use negative integers:
----------------------------
>>> add_integer(2, -1)
1
>>>

>>> add_integer(-1, 2)
1
>>> 

>>> add_integer(-1, -2)
-3
>>> 



We can use float:
-----------------
>>> add_integer(1, 1.5)
2
>>>

>>> add_integer(1.5, 1)
2
>>> 

>>> add_integer(1.5, 1.5)
2
>>> 


We can use Negative float
-------------------------
>>> add_integer(1.5, -1.5)
0

>>> add_integer(-1.5, 1.5)
0

>>> add_integer(-1.5, -1.5)
-2
>>>


We can Perform Some math operations with individual parameters
--------------------------------------------------------------
addition:
_________
>>> add_integer(-1.5, 5+5)
9
>>> 

minus:
______
>>> add_integer(-1.5, 5*5)
24
>>> 

division:
_________
>>> add_integer(5/5, 5)
6
>>>

mod:
___
>>> add_integer(5%5, 5)
5
>>>

multiplications:
_______________
>>> add_integer(5*5, 5)
30
>>> 


These Cases will give errors:
=============================

With strings:
____________
>>> add_integer(5, 'hello')
Traceback (most recent call last):
TypeError: b must be an integer
>>> 

>>> add_integer('hello', 5)
Traceback (most recent call last):
TypeError: a must be an integer
>>>

With A None argument:
____________________
>>> add_integer(None)
Traceback (most recent call last):
TypeError: a must be an integer
>>>
